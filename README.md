# AirKeyBoard

A keyboard that allows users to type in the air.

### Environment

* Leap motion
* Java SDK 10

### About

* specially designed user interface
* can recognize typing gesture.
* allows 2 gestures: delete and move cursor.
* can correct small spelling errors.
